# Reads a sybil model file and constructs an object of cobrar's class 'ModelOrg'

Reads a sybil model file and constructs an object of cobrar's class
'ModelOrg'

## Usage

``` r
readSybilmod(file_path)
```

## Arguments

- file_path:

  Path to sybil model(s) saved as RDS file.

## Value

A [ModelOrg-class](ModelOrg-class.md) object or a list with
[ModelOrg-class](ModelOrg-class.md) objects.
