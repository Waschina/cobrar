# Set objective direction

Set the objective direction to either maximize or minimize

## Usage

``` r
setObjDir(model, dir)
```

## Arguments

- model:

  Model of class
  [ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md)

- dir:

  Character. Either "maximize" or "minimize".

## Value

An updated model of class
[ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md)
