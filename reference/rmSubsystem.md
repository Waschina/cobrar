# Remove subsystems from a model

This function removes specified subsystems from a model.

## Usage

``` r
rmSubsystem(model, subsystem)
```

## Arguments

- model:

  Model of class
  [ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md)

- subsystem:

  A character vector stating the subsystem IDs in a model or a numeric
  vector providing the subsystem indices.

## Value

An updated model of class
[ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md)
