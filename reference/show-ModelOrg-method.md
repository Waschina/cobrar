[ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md).

## Usage

``` r
# S4 method for class 'ModelOrg'
show(object)
```

## Arguments

- object:

  S4-object of class
  [ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md).
