# Solve an LP problem

Solves an
[LPproblem](https://waschina.github.io/cobrar/reference/LPproblem-class.md).

## Usage

``` r
solveLp(lp, ...)

# S4 method for class 'LPproblem_glpk'
solveLp(lp)
```

## Arguments

- lp:

  Object of class
  [LPproblem](https://waschina.github.io/cobrar/reference/LPproblem-class.md)

- ...:

  Additional parameters passed on to the specific method instance.
