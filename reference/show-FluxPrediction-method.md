# Print a short summary of a flux prediction result

Displays a summary of the results obtained from a metabolic flux
prediction (e.g., FBA or pFBA).

## Usage

``` r
# S4 method for class 'FluxPrediction'
show(object)
```

## Arguments

- object:

  S4-object of class [FluxPrediction](FluxPrediction-class.md).
