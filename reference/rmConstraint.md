# Remove constraints

Remove specific user constraints from a metabolic model.

## Usage

``` r
rmConstraint(model, ind)
```

## Arguments

- model:

  Model of class
  [ModelOrg](https://waschina.github.io/cobrar/reference/ModelOrg-class.md)

- ind:

  Integer vector with the indices of the constraints to be removed
